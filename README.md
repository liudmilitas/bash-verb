# Bashkir Grammar Practice App (WIP)

### Web app for practicing Bashkir verb conjugations
Created with Next.js, TypeScript and Tailwind

### Quick Setup

```bash
# install dependencies
$ npm install

# build for development
$ npm run dev

# build for production
$ npm run build
```
